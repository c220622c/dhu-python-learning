在本次作业中,如果我们想要重复利用函数,该如何进行?
假设我们想要直接使用`230250329_9_3.py`里的`countSelectedFruit`

```python
import 230250329_9_3
```

这会返回

```terminal
File "/Users/wzj/Documents/Coding-repo/dhu-python/230250329_9/230250329_9_4.py", line 2
    import 230250329_9_3
           ^^^^^^^^^^^^^
SyntaxError: invalid syntax
```

这是因为 python 编辑器要求导入的模块不能以数字开头
这时我们就需要使用`importlib.util`

```python
import importlib.util

# 导入模块
spec = importlib.util.spec_from_file_location("module9_3", "230250329_9_3.py")
module9_3 = importlib.util.module_from_spec(spec)
spec.loader.exec_module(module9_3)
def sth():
    print("This is a placeholder function.")
```

此时我们运行函数,会发现 python 导入函数的时候,会先运行一遍

